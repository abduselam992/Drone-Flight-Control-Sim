# Drone PID Flight Controller Simulation

## Project Goal
To demonstrate how a PID (Proportional-Integral-Derivative) algorithm controls a UAV's altitude by balancing thrust against gravity.

## Technical Details
* **Language:** Python
* **Concepts:** Linear Control Systems, Newtonian Physics, PID Tuning.
* **Output:** A visual graph showing the drone taking off and stabilizing at a 10m hover.

## What I Learned
Through this project, I learned how flight controllers (like ArduPilot or PX4) use calculus to maintain stability and how 'gains' (Kp, Ki, Kd) affect the responsiveness of a drone.
